### waipahu
=====

An OTP application

Build
-----

    $ rebar3 new umbrella waipahu

Support Elixir's function
-----

    >  waipahu_app:to_decimal(1).
    1.0
    > waipahu_app:last([1, 2, 3]).
    3
    > waipahu_app:downcase(<<"Aloha!">>)
    aloha!
